# ماه‌چین - MahChin

اپلیکیشن اندرویدی فارسی و راست‌چین برای برنامه‌ریزی ماهانه، پیگیری روزانه، پروژه‌ها، آلارم اختصاصی و مایندمپ.


## تغییرات نسخه ۱.۹.۰

- تسک‌هایی که از مایندمپ ساخته می‌شوند در صفحه امروز به شکل جمع‌شونده نمایش داده می‌شوند؛ اول پروژه، بعد شاخه اصلی، بعد زیرشاخه‌ها.
- سرشاخه اصلی و زیرشاخه‌ها قابل تیک خوردن شدند؛ با تیک سرشاخه، همه تسک‌های زیرمجموعه همان بخش انجام‌شده می‌شوند و با برداشتن تیک، همه برمی‌گردند.
- تبدیل مایندمپ به تسک، ساختار تو‌در‌تو را حفظ می‌کند تا معلوم باشد هر تسک مربوط به کدام پروژه و کدام شاخه است.
- حذف شاخه مایندمپ حالا کل زیرشاخه‌های داخل آن را هم به‌صورت امن حذف می‌کند.
- حالت تقسیم مایندمپ به تسک همچنان با انتخاب تاریخ شروع و تعداد تسک در روز کار می‌کند؛ مثلاً روزی ۱ کار یا روزی ۵ کار.
- نمایش مایندمپ و تسک‌ها خلوت‌تر شد تا صفحه برای پروژه‌های شلوغ ذهن را درگیر نکند.

## تغییرات نسخه ۱.۸.۰

- متن دکمه‌های پایین مایندمپ دیگر به خط بعد نمی‌رود؛ «شاخه»، «زیرشاخه»، «ویرایش» و «تسک» همیشه یک‌خطی نمایش داده می‌شوند.
- مدیریت پروژه‌ها ساده‌تر شد؛ از دکمه «پروژه» می‌توان پروژه جدید ساخت، پروژه فعلی را ویرایش کرد یا حذف کرد.
- حذف پروژه به‌صورت امن انجام می‌شود و اگر فقط یک پروژه باقی مانده باشد، پروژه آخر حذف نمی‌شود.
- برای خلوت ماندن صفحه، گزینه‌های مدیریت پروژه داخل یک منوی پایین صفحه قرار گرفتند.

## تغییرات نسخه ۱.۷.۰

- رابط کاربری مایندمپ شبیه‌تر به تجربه موبایل XMind شد.
- مایندمپ از حالت سخت و محدود خارج شد و به فضای آزاد قابل حرکت و زوم تبدیل شد.
- می‌توانی با کشیدن صفحه در مایندمپ حرکت کنی و با دو انگشت زوم کنی.
- نودها به شکل کارت‌های رنگی شاخه‌ای نمایش داده می‌شوند، نه دایره‌های سخت و فشرده.
- هر شاخه اصلی رنگ پررنگ و متفاوت دارد و زیرشاخه‌ها همان خانواده رنگی را حفظ می‌کنند.
- با لمس هر نود، همان نود انتخاب می‌شود و دکمه «زیرشاخه» پایین صفحه فعال می‌شود.
- برای تعریف سریع‌تر، پایین صفحه دکمه‌های ثابت «شاخه»، «زیرشاخه»، «ویرایش» و «تسک» اضافه شد.
- با نگه‌داشتن روی نود، منوی افزودن زیرشاخه، افزودن شاخه کنار این، ویرایش و حذف باز می‌شود.
- دکمه «مرکز» برای برگشت سریع به مرکز مایندمپ اضافه شد.

## تغییرات نسخه ۱.۶.۰

- مایندمپ واقعی رنگی‌تر و هنری‌تر شد.
- هر شاخه اصلی مایندمپ یک رنگ پررنگ و متفاوت دارد.
- زیرشاخه‌های هر شاخه با همان خانواده رنگی نمایش داده می‌شوند تا در پروژه‌های شلوغ گم نشوند.
- خط اتصال هر شاخه هم‌رنگ همان شاخه شد.
- برای پروژه‌های شلوغ، چیدمان شاخه‌های اصلی فشرده‌تر و خواناتر شده است.

## تغییرات نسخه ۱.۵.۰

- تب مایندمپ از حالت لیستی خارج شد و به **مایندمپ واقعی گره‌ای** تبدیل شد.
- مایندمپ شبیه نمونه تصویری است: نود مرکزی پروژه، شاخه‌های اصلی دور آن و زیرشاخه‌ها با خط اتصال.
- با نگه‌داشتن روی هر نود، منوی خلوت باز می‌شود:
  - افزودن زیرشاخه
  - ویرایش
  - حذف
- با نگه‌داشتن روی نود مرکزی یا لمس فضای خالی، می‌توان شاخه اصلی جدید ساخت.
- دکمه پایین صفحه برای افزودن شاخه اصلی و تبدیل مایندمپ به تسک باقی مانده است تا صفحه شلوغ نشود.
- تبدیل مایندمپ به تسک همچنان فعال است و زیرشاخه‌های نهایی را طبق تعداد روزانه در تقویم پخش می‌کند.
- نسخه اپ به `1.5.0` و `versionCode = 6` ارتقا داده شد.

## تغییرات نسخه ۱.۴.۰

- پروژه‌ها اضافه شدند؛ هنگام افزودن تسک می‌توان پروژه را از کشویی انتخاب کرد.
- آلارم اختصاصی برای هر تسک اضافه شد.
- آلارم از صدای پیش‌فرض Alarm گوشی استفاده می‌کند.
- ورود تسک با ویس اضافه شد.
- نسخه اولیه تب مایندمپ و تبدیل مایندمپ به تسک اضافه شد.

## تغییرات نسخه ۱.۳.۰

- لیست تسک‌های امروز خلوت‌تر و شبیه Todo شد.
- با زدن تیک هر تسک، انجام‌شده می‌شود؛ اگر دوباره بزنی، تیک برداشته می‌شود.
- گزینه‌های شلوغ مثل ویرایش، حذف، انتقال به فردا، انتقال به تاریخ دلخواه و لغو با نگه‌داشتن روی تسک یا سه‌نقطه باز می‌شوند.
- قالب ماهانه هم به لیست Todo خلوت تبدیل شد.
- دکمه افزودن تسک به‌صورت بزرگ و ثابت پایین صفحه قرار گرفت.
- متن نوتیفیکیشن ساده‌تر شد و فقط کارهای باز امروز را نشان می‌دهد.

## امکانات

- Room Database آفلاین
- WorkManager برای یادآوری دوره‌ای
- AlarmManager برای آلارم اختصاصی تسک‌ها
- تقویم شمسی داخلی بدون وابستگی خارجی
- پشتیبانی سال‌های ۱۴۰۵ تا ۱۵۰۰
- قالب ماهانه تکرارشونده
- تسک اختصاصی برای تاریخ خاص
- ساخت خودکار برنامه امروز از قالب ماهانه
- پروژه‌ها و دسته‌بندی تسک‌ها بر اساس پروژه
- مایندمپ واقعی گره‌ای برای خرد کردن پروژه‌ها
- تبدیل مایندمپ به تسک‌های روزانه
- ورود عنوان تسک با ویس
- انتقال تسک‌های روز ۳۱ به آخرین روز ماه‌های کوتاه‌تر
- انتقال روز ۳۰ اسفند به روز ۲۹ در سال غیرکبیسه
- وضعیت‌ها: انجام نشده، در حال انجام، انجام شد، موکول به فردا، موکول به تاریخ دلخواه، لغو شد
- اولویت‌ها: عادی، مهم، فوری
- تنظیم شدت یادآوری، ساعت شروع و پایان، صدا، ویبره و حالت تاریک

## گرفتن APK آنلاین با GitHub Actions

1. فایل‌های پروژه را داخل Repository آپلود کن.
2. برو تب `Actions`.
3. روی `Build MahChin APK` بزن.
4. `Run workflow` را اجرا کن.
5. بعد از سبز شدن، از بخش `Artifacts` فایل APK را دانلود کن.

قبل از نصب نسخه جدید، اگر نسخه قبلی روی گوشی نصب است، آن را حذف کن.
