# ماه‌چین - MahChin

اپلیکیشن اندرویدی فارسی و راست‌چین برای برنامه‌ریزی ماهانه، پروژه‌ای و پیگیری روزانه.

## تغییرات نسخه ۱.۴.۰

- اضافه شدن پروژه‌ها؛ هر تسک می‌تواند از طریق کشویی به یک پروژه وصل شود.
- اضافه شدن تب جدید «مایندمپ» برای خرد کردن هدف‌های یک پروژه به زیرمجموعه‌ها.
- امکان تبدیل زیرمجموعه‌های مایندمپ به تسک‌های چند روز، مثلاً روزی ۵ تسک تا تمام شود.
- ذخیره مسیر هدف مایندمپ داخل تسک؛ مشخص می‌شود تسک زیرمجموعه کدام هدف است.
- اضافه شدن آلارم اختصاصی برای هر تسک با صدای پیش‌فرض آلارم گوشی.
- اضافه شدن آلارم ماهانه برای تسک‌های قالب ماهانه.
- اضافه شدن ورود تسک با ویس؛ در پنجره افزودن تسک روی آیکون میکروفون بزن و عنوان را بگو.
- حفظ ظاهر خلوت Todo؛ گزینه‌های شلوغ همچنان با نگه‌داشتن روی تسک یا سه‌نقطه باز می‌شوند.

## امکانات اصلی

- Kotlin + Jetpack Compose
- Room Database آفلاین
- WorkManager برای یادآوری دوره‌ای
- AlarmManager برای آلارم اختصاصی تسک‌ها
- تقویم شمسی داخلی بدون وابستگی خارجی
- پشتیبانی سال‌های ۱۴۰۵ تا ۱۵۰۰
- قالب ماهانه تکرارشونده
- تسک اختصاصی برای تاریخ خاص
- پروژه‌بندی تسک‌ها
- مایندمپ پروژه و تبدیل مایندمپ به تسک
- ورود صوتی تسک با Speech Recognizer گوشی
- وضعیت‌ها: انجام نشده، در حال انجام، انجام شد، موکول به فردا، موکول به تاریخ دلخواه، لغو شد
- اولویت‌ها: عادی، مهم، فوری
- تنظیم شدت یادآوری، ساعت شروع و پایان، صدا، ویبره و حالت تاریک

## گرفتن APK آنلاین با GitHub Actions

1. فایل‌های پروژه را داخل Repository آپلود کن.
2. برو تب `Actions`.
3. روی `Build MahChin APK` بزن.
4. `Run workflow` را اجرا کن.
5. بعد از سبز شدن، از بخش `Artifacts` فایل `MahChin-release-apk` را دانلود کن.

قبل از نصب نسخه جدید، اگر نسخه قبلی روی گوشی نصب است، آن را حذف کن.
