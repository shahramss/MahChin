# ماه‌چین - MahChin

اپلیکیشن اندرویدی فارسی و راست‌چین برای برنامه‌ریزی ماهانه و پیگیری روزانه.

## تغییرات نسخه ۱.۱

- طراحی تیره، ساده‌تر و مدرن‌تر
- صفحه امروز مینیمال‌تر و متمرکزتر
- کارت‌های تسک خلوت‌تر
- فیکس دکمه ذخیره تنظیمات یادآوری
- افزودن پیام ذخیره تنظیمات
- افزودن نوتیفیکیشن آزمایشی در تنظیمات
- زمان‌بندی سریع یادآوری بعد از ذخیره، افزودن یا تغییر وضعیت تسک
- خروجی Release امضاشده آزمایشی از GitHub Actions
- compileSdk = 37

## امکانات

- Kotlin + Jetpack Compose
- MVVM ساده
- Room Database آفلاین
- WorkManager برای یادآوری دوره‌ای
- تقویم شمسی داخلی بدون وابستگی خارجی
- پشتیبانی سال‌های ۱۴۰۵ تا ۱۵۰۰
- قالب ماهانه تکرارشونده
- تسک اختصاصی برای تاریخ خاص
- ساخت خودکار برنامه امروز از قالب ماهانه
- انتقال روز ۳۱ به آخرین روز ماه‌های کوتاه‌تر
- وضعیت‌ها: انجام نشده، در حال انجام، انجام شد، موکول، لغو شد
- اولویت‌ها: عادی، مهم، فوری
- تنظیم شدت یادآوری، ساعت شروع/پایان، صدا، ویبره، حالت تاریک

## گرفتن APK آنلاین

1. پروژه را در GitHub آپلود کن.
2. برو به تب **Actions**.
3. workflow با نام **Build MahChin APK** را اجرا کن.
4. از **Artifacts** فایل `MahChin-release-apk` را دانلود کن.
5. ZIP را باز کن و `app-release.apk` را روی گوشی نصب کن.

## نکته نوتیفیکیشن

در Android 13 به بعد باید مجوز Notification را بدهی. در صفحه تنظیمات، روی «ارسال نوتیفیکیشن آزمایشی» بزن. اگر پیام مجوز داد، از تنظیمات گوشی اجازه نوتیفیکیشن را فعال کن.

برای بعضی گوشی‌ها، اگر Battery Optimization خیلی سخت‌گیر باشد، یادآوری‌های پس‌زمینه ممکن است دیرتر نمایش داده شوند.
